================================================================================
                    SECURE FILE SHARING SYSTEM - UI DESCRIPTION
================================================================================

This document describes the user interface (UI) for each module of the Secure
File Sharing System based on the current HTML/CSS implementation.

================================================================================
                              GENERAL STYLING
================================================================================

Theme & Colors:
- Dark theme with slate/dark blue background (bg-slate-950)
- Grid pattern overlay on background
- Glass-morphism card effects (backdrop blur, semi-transparent backgrounds)
- Gradient accent colors: Blue (#3b82f6) to Emerald (#34d399)
- Text colors: White/slate-100 for primary, slate-300/400 for secondary

Common Elements:
- Rounded corners (rounded-xl, rounded-2xl, rounded-3xl)
- Lifted card effects with subtle shadows
- Consistent spacing using Tailwind-like utility classes
- Responsive design (mobile-first with sm/md/lg breakpoints)

  5. Timestamp (when it occurred)

- Appear in corner of screen
- Types: success (ok), error (err), info
- Auto-dismiss after timeout


================================================================================
                              RESPONSIVE DESIGN
================================================================================

Mobile (< 640px):
- Single column layouts
- Hamburger menu (hidden nav links)
- Stacked form fields
- Full-width buttons
- Hidden user info in header

Tablet (640px - 1024px):
- 2-column grids
- Visible navigation
- Side-by-side form layouts

Desktop (> 1024px):
- Full multi-column grids (up to 4 columns)
- Expanded navigation
- Wide content containers (max-w-6xl)


================================================================================
                              ACCESSIBILITY
================================================================================

- Skip to content link
- ARIA labels on interactive elements
- Role attributes on tab components
- Semantic HTML structure (header, main, section, etc.)
- Keyboard navigation support
- Focus indicators on interactive elements
